# Zodiac Cards Collection
**Credits:** Original concept by [u/Euuphorix on Reddit](https://www.reddit.com/r/balatro/s/G2NwhYZrnh)

Zodiac cards are a new set of collectable items that provide unique persistent effects triggered by specific poker hands.

| Name | Effect |
| :--- | :--- |
| **Aquarius** | If hand is **Two Pair**, converts the lower-ranking pair into the higher-ranking pair (permanently transforming the cards). |
| **Pisces** | If hand is **Royal Flush**, upgrades **Straight Flush** level by 3. |
| **Aries** | If hand is **Four of a Kind**, played cards permanently gain **+15 Chips**. |
| **Taurus** | If hand contains **Three of a Kind** and three unique suits, gives **X2 Mult**. |
| **Gemini** | If hand is **Pair** and cards have the same suit, destroys played cards. |
| **Cancer** | If hand is **Flush House**, doubles the **Base Chips** and **Base Mult** for the current hand. |
| **Leo** | If hand is **Five of a Kind**, 1 in 20 chance for played cards to become **Foil** or **Holographic**. |
| **Virgo** | If hand is **Full House**, earn **$5** when played. |
| **Libra** | If hand is **Flush Five**, 1 in 30 chance for played cards to become **Polychrome**. |
| **Scorpio** | If hand is **High Card**, played cards permanently gain **+2 Mult**. |
| **Sagitarius** | If hand is **Flush**, converts one random card in your deck to match the suit of the played hand. |
| **Capricorn** | If hand is **Straight**, played **Aces** permanently gain **X0.1 Mult**. |
